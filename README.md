# MicroserviceAPP--Products

## Project setup

```bash
$ npm install
```

## first steps
1. run ```npm i```
2. create `.env` base on `.env.template`
3. run prisma `npx prisma migrate dev`

### Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
